# ⚾ Python 야구 감독 시뮬레이션 (Baseball Manager Sim)

이 프로젝트는 객체 지향 프로그래밍(OOP)을 활용하여 개발된 텍스트 기반 야구 시뮬레이션 게임입니다. 사용자는 직접 구단의 감독이 되어 팀 이름, 선수 명단, 스탯을 설정하고, 매 타석마다 작전(강공, 도루, 번트)을 지시하며 경기를 진행할 수 있습니다.

## ✨ 주요 기능

* **커스텀 팀/선수 생성**: 팀 이름 및 9명의 야수, 1명(선발) 이상의 투수 스탯을 직접 설정 가능.
* **스탯 기반 시뮬레이션**: 타자의 컨택/파워, 투수의 제구/구위에 따라 타석 결과가 역동적으로 결정됨.
* **감독 자유도**: 유저의 작전 지시(도루, 번트)를 통해 경기 흐름에 개입 가능.
* **돌발 이벤트**: 경기 중 발생하는 랜덤 이벤트로 변수와 재미를 추가.
* **모듈화된 코드**: `models.py`, `game_engine.py`, `main.py`로 분리된 깔끔한 구조.

## 🛠️ 기술 스택

* **언어**: Python 3.x
